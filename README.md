# Getting Started
## Prerequisites
* npm 
```
npm install npm@latest -g

```
* node 
```
Node 16.14

```

## Installation
1. Clone the repo

```
git clone https://github.com/crthai/project-form.git

```
2. Install NPM packages

```
npm install 

```

## Usage
### Run
```
npm start

```
### Test 
```
npm run test:e2e and npm test:cov

```
